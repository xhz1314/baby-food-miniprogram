# 宝宝餐菜谱库

## 项目简介

`宝宝餐菜谱库` 是一个面向家庭日常制作场景的微信小程序项目，提供可直接查看的宝宝餐菜谱内容。

项目定位为**菜谱查询与收藏工具**，核心能力是展示静态菜谱列表、食材清单、做法步骤，以及常用菜谱收藏，方便家长按季节查找合适的家常宝宝餐做法。

## 功能范围

- 浏览宝宝餐菜谱列表
- 按季节筛选菜谱
- 按分类筛选菜谱（主食、蛋白质、蔬菜、汤羹）
- 查看食材清单与做法步骤
- 收藏与取消收藏常用菜谱
- 本地保存收藏记录

## 当前内容特点

- 菜谱按季节整理，展示常见家常宝宝餐做法
- 菜谱内容为项目内置静态数据，不包含用户投稿内容
- 做法步骤会结合宝宝成长阶段调整食材处理方式
- 首页展示为菜谱列表与筛选，不包含资讯流、文章发布或社区互动

## 页面说明

- `pages/index/index`：首页，展示菜谱列表与筛选
- `pages/detail/detail`：详情页，展示食材与做法步骤
- `pages/favorite/favorite`：收藏页，查看已收藏菜谱

## 适用说明

适合用于宝宝餐菜谱工具类微信小程序的原型开发、界面实现与功能演示。
