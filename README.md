# 宝宝今日食

## 项目简介

`宝宝今日食` 是一个面向婴幼儿辅食场景的微信小程序，用于为家长提供按月龄阶段与季节特点生成的每日辅食搭配建议。

项目围绕“今日吃什么”这一核心需求，结合宝宝当前成长阶段、食材口感处理建议以及当季蔬菜信息，生成更适合家庭日常参考的菜单内容。当前首页会展示一组包含主食、优质蛋白和当季蔬菜的推荐菜单，并支持下拉切换不同组合。

除了每日推荐外，小程序还提供菜谱详情查看与收藏能力。用户可以查看每道辅食的食材、步骤、适龄说明和处理建议，也可以将感兴趣的菜谱加入收藏，方便后续反复查看。收藏内容使用本地存储保存，并保留快照信息，尽量避免因后续数据变化导致已收藏内容丢失。

## 当前功能

- 按宝宝月龄生成适龄辅食推荐
- 结合上海时令蔬菜生成每日菜单
- 展示主食、蛋白质、蔬菜的组合方案
- 支持菜谱详情查看
- 支持菜谱收藏与取消收藏
- 支持收藏记录本地持久化

## 页面组成

- `pages/index/index`：首页，展示今日推荐菜单
- `pages/detail/detail`：详情页，展示菜谱做法与喂养建议
- `pages/favorite/favorite`：收藏页，查看已收藏菜谱

## 适用场景

适合用于宝宝辅食推荐类、母婴饮食参考类微信小程序的原型开发与功能演示。
